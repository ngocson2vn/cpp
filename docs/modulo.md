# Modulo vs Modulus
In mathematics and programming, **modulo** is the operation or function itself, while **modulus** is the specific number used as the divisor in that operation.

For example, in the expression $13 \pmod{5} = 3$, you are performing a modulo operation using a modulus of 5 to find the remainder.

## Common rules
For any positive integers $a$ and $p$, then <br/>
$a \pmod{2^p} = a\ \text{AND}\ (2^p - 1)$
